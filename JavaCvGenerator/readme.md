#Generateur de cv
Permet de générer un cv au format txt, word et pdf à partir d'un cv xml.
Certains drh ou sociétés demandent de remplir un cv à un format particulier.  
C'est particulièrement saoulant.  
Cet outil devrait permettre de faciliter la réponse à de telles demandes.  


#Pour utiliser le générateur:  
Modifier le fichier cv_bg.xml  puis java bg.cv.Main  
ou  
java bg.cv.Main /pathToXmlFile
